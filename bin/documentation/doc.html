1. Introduction

Ce logiciel permet de lire et afficher l'heure sur une image de montre analogique. Il est conçu pour fonctionner sous Windows.

3. Comment utiliser le logiciel

Après le lancement, cliquez sur File > Open image pour ouvrir une image. Tous les formats sont pris en charge.

Une fois l'image ouvert, cliquez sur Read hour pour lancer le traitement et afficher l'heure. L'heure détectée s'affiche en dessous de l'image. de base le texte est à 00:00

Si le traitement c'est bien déroulé, le bouton Detect clock s'active. En cliquant dessus vous pouvez voir l'image resultat sur laquelle on fait la détection de chaque aiguille.

C'est possible de voir les images présentant les étapes du traitement qui est fait. Il suffit de cliquer sur les fléches gauche ou droite de la barre d'outils.

Les autres boutons de la barre d'outils permettent de faire un zoom ou dezoom dans l'image

La barre de status donne des informations sur la position du curseur sur l'image et le status du traitement ou l'aide lorsqu'on survol les boutons
