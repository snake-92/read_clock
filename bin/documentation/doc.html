1. Introduction

Ce logiciel permet d'automatiser le traitement des images et d'interagir avec une interface conviviale. Il est conçu pour fonctionner sous Windows avec GCC et prend en charge wxWidgets et OpenCV.

2. Installation

Téléchargez la dernière version depuis notre dépôt GitHub. Décompressez et exécutez setup.exe. Assurez-vous que TDM-GCC est installé.

3. Comment utiliser le logiciel

Après le lancement, cliquez sur Fichier > Nouveau Projet pour démarrer. Utilisez les outils dans la barre pour charger une image et appliquer des filtres.
